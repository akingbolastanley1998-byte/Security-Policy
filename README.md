## Available Projects

### [Enterprise Security Policy Framework](./Enterprise-Security-Policy-Portfolio.md)

**Client Sector:** Financial Services (UK)  
**Project Type:** Enterprise Security Architecture & Policy Design  
**Duration:** 6 weeks  
**Role:** Lead Security Consultant  

**Project Overview:**

Comprehensive security policy framework designed for a mid-tier UK financial institution serving 250,000+ customers. The project addressed regulatory requirements across PCI DSS v4.0, UK GDPR, FCA operational resilience, and ISO/IEC 27001:2022.

**Key Deliverables:**
- Enterprise risk assessment using STRIDE threat modeling
- Network Security Policy with governance framework
- Technical Security Standards (segmentation, cryptography, monitoring)
- Operational Security Procedures (incident response, change control)
- Three-zone defense-in-depth network architecture

**Regulatory Compliance:**
- PCI DSS v4.0 - Cardholder Data Environment design
- UK GDPR / DPA 2018 - Data protection controls
- FCA SYSC 8 / DORA - Operational resilience requirements
- ISO/IEC 27001:2022 - Information security management

**Skills Demonstrated:**
- Security architecture design (defense-in-depth, zone segmentation)
- Policy and procedure documentation
- Risk assessment and threat modeling
- Regulatory compliance (PCI DSS, GDPR, FCA)
- Network security design (ACLs, VPN, SIEM)
- Cryptographic standards (TLS, IPsec)
- Change control and governance frameworks

**Technical Components:**
- Three-zone network segmentation (Access, Distribution, Core Operations)
- Default-deny inter-zone traffic matrix
- Site-to-site IPsec VPN for disaster recovery
- Centralized SIEM logging (Splunk Enterprise Security)
- Four-eyes change control procedures
- Multi-factor authentication requirements
